Futr Flatpak Repository

Stable Repository

To add the stable repository:

flatpak remote-add --if-not-exists --no-gpg-verify futr-stable https://futrnostr.github.io/futr-flatpak/stable

Or download the repository file

Continuous Repository

To add the continuous repository:

flatpak remote-add --if-not-exists --no-gpg-verify futr-continuous https://futrnostr.github.io/futr-flatpak/continuous

Or download the repository file
